## 目标

本章将 MDN 中的 WebRTC API 文档的英文文档 翻译为中文文档。

- 英文基础较好的，建议直接看英文文档。
- 这部分已经有中文文档，为什么还要再翻译一遍？MDN 上中文文档虽然已经翻译的比较好，但是相对于英文文档，有较多内容丢失或者缺乏翻译，因此另做翻译，已做梗概学习和便捷翻阅之用。
- 此文档对源文档有所扩展；部分语句有人工润色，可能与源文档意思有所偏差，敬请指正。
